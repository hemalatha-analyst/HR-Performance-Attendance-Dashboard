# HR Employee Performance & Attendance Dashboard

## 📌 Objective
To analyze employee performance, attendance, and productivity using Excel.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard

## 📊 Features
- KPI tracking (Employees, Performance, Attendance)
- Department-wise performance analysis
- Attendance distribution
- Monthly performance trends
- Interactive dashboard with slicers

## 📈 Key Insights
- Identified high and low performers
- Analyzed attendance patterns
- Compared department performance
